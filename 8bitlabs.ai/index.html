What this is

A public lab notebook for Solana-native AI.

8 Bit Labs packages the work inside ai-training/ into a public product surface: datasets, model cards, LoRA configs, NVIDIA blueprints, perps tools, model-kit workflows, Hugging Face releases, W&B tracked jobs, Ollama artifacts, and devnet registry transactions.

The goal is simple: make the training stack legible enough that a builder can inspect the data, run the pipeline, use the models, and verify the first cohort on Solana.

Clawd Grok Computer

Run Clawd Grok inside a disposable sandbox computer.

The launch surface packages core-ai/clawd-grok into an E2B computer, clones the repo, installs dependencies, builds the Grok terminal runtime, and runs a safe status smoke command. Production execution is gated by server-side E2B credentials and sandbox policy.

What we built

A reproducible AI training and registry stack.

01

Collect and normalize

Prepare message JSONL, CPT transaction rows, PDF/notebook/parquet research, perps tools, and Clawd source chunks.

02

Clean and split

Deduplicate, filter secret-like records, enforce public-safe artifacts, and produce deterministic train/eval/test splits.

03

Train and track

Run LoRA training on local Mac sanity passes or remote HF A100 jobs, with W&B tracking for metrics and traces.

04

Package and serve

Publish Hugging Face datasets and adapters, generate Ollama modelfiles, and route inference through the mesh.

05

Register and attest

Register model accounts to the Solana devnet program and attach SAS provenance metadata for public inspection.
